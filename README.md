# Контакты DEMO

Демо-версия прототипа мобильного приложения «Менеджер контактов» для Android 5.0+.
Проект сделан как нативное приложение (Kotlin + AndroidX) и повторяет ключевые функции HTML-прототипа:
- список контактов
- поиск
- фильтрация по группам
- добавление/редактирование/удаление
- признак «рабочий контакт» и задача
- простой импорт из текста

## Быстрый запуск
1. Откройте проект в Android Studio или VS Code.
2. Создайте файл `local.properties` с путем к Android SDK.
3. Запустите сборку через Gradle.

## Структура
- `app/src/main/java/.../data` — модель и работа с SQLite
- `app/src/main/java/.../ui` — экраны и адаптер списка
- `app/src/main/res/layout` — разметка интерфейса
- `docs` — описание будущего приложения (внешняя часть и бэкэнд)

## Примечания
- Это DEMO. Функции синхронизации, экспорта, продвинутого импорта и резервного копирования здесь описаны, но не реализованы.
