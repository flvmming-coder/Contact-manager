Описание версии: Contact Manager v0.12.2a (TEST)
Тип: БЭКЭНД

Причина обновления:
Нужно диагностировать причины краша при старте и получить воспроизводимые логи непосредственно на телефоне.

Что добавлено:
1. Глобальный логгер приложения (AppEventLogger).
2. Логирование ключевых событий:
- старт приложения,
- загрузка и рендер контактов,
- валидации,
- операции сохранения/удаления.
3. Перехват необработанных исключений через default UncaughtExceptionHandler с записью stack trace.
4. Лог-файлы создаются на каждый запуск в директории:
`Android/data/com.example.contactmanagerdemo/files/Documents/ContactManagerLogs/`
5. Добавлена диагностика поврежденных сохраненных данных контактов:
- при ошибке парсинга JSON пишется ошибка в лог,
- при невалидных данных автоматически подставляются стартовые контакты.

Статус проверки:
- локально: сборка, lint и unit-тесты проходят,
- запуск на эмуляторах Android 6–14 в текущей среде не выполнен из-за отсутствия adb/emulator/AVD.
