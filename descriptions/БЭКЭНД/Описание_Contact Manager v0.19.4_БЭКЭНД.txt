Contact Manager v0.19.4 (34)
БЭКЭНД

Причина обновления:
- расширение инструментов администрирования и актуализация данных номера в зеркале БД.

Изменения логики:
1) Экспорт БД:
- добавлен генератор `.xlsx` (OpenXML, без внешних сервисов);
- реализована отправка файла из приложения через `FileProvider`.

2) Зеркало SQLite:
- добавлено поле `phone_raw` для номера без маски в формате `+7XXXXXXXXXX` (РФ/КЗ);
- обновлён snapshot-вывод БД.

3) Поиск:
- сравнение идёт по нескольким представлениям номера:
  - исходная строка с маской;
  - компактный вариант без разделителей;
  - набор только цифр;
  - `phone_raw` для РФ/КЗ.

4) Сортировка:
- внедрены режимы сортировки по имени и по времени создания.

5) Админ-панель:
- интервал long-press для открытия уменьшен до 2 секунд.
