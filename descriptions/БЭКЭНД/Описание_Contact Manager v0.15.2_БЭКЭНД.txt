Описание версии Contact Manager v0.15.2 (18) — БЭКЭНД

Причина обновления:
Требовалась доработка бизнес-логики: разграничить права на ручную настройку цвета, расширить импорт контактов фото-данными и корректно обрабатывать многочастные имена.

Что изменено в логике:
1. Добавлен флаг в dev-настройках для ручного редактирования цвета аватара:
- хранится в `contact_manager_dev_settings`;
- управляется из админ-панели.
2. В диалоге редактирования контакта:
- при выключенном dev-флаге ручной выбор цвета скрыт;
- для обычного режима сохраняется только случайный цвет (через `avatarColor = null`).
3. Импорт контактов с устройства:
- добавлен импорт `PHOTO_URI` в модель контакта (`avatarPhotoUri`), если фото существует;
- расширена структура `ImportedContactDetails`.
4. Обработка имен:
- убрано обрезание имени до первого слова при импорте;
- сохранение полных строк имени/фамилии из источника.
5. Добавлен отдельный диалог Developers:
- реализованы кликабельные `ClickableSpan` для перехода на VK-профили.
6. Доработана адаптация цветов для тёмной темы в ключевых диалогах.
7. Обновлены версии:
- `versionName = 0.15.2`
- `update_sequence = 18`

Проверка:
Сборка, lint и unit-test выполнены успешно.
