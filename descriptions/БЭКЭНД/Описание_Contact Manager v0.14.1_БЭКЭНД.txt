Описание версии: Contact Manager v0.14.1 (TEST)
Тип: БЭКЭНД

Причина обновления:
Нужно добавить управляемое логирование, проверку новых релизов и хранение дополнительного текстового поля контакта.

Что изменено в логике/данных:
1. Расширена модель Contact:
- добавлено поле comment.
2. Хранилище ContactPrefsStorage:
- сохранение/чтение comment,
- сохранена обратная совместимость со старыми данными.
3. MainActivity:
- добавлена валидация комментария (максимум 512 символов),
- реализован скрытый триггер админ-панели (5 тапов/2 сек),
- реализована проверка сети,
- реализован запрос к GitHub API releases/latest и сравнение версий.
4. AppEventLogger:
- добавлен переключатель enabled/disabled с сохранением настройки,
- добавлен метод получения текущего пути логов,
- попытка записи логов в Android/data/com.contact.manager/ContactManagerLogs,
- fallback на app-specific путь при ограничениях доступа.

Технический статус:
- assembleDebug: успешно,
- lintDebug: успешно,
- testDebugUnitTest: успешно.
