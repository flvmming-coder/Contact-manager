# Описания версий

Папка для централизованного хранения всех описаний проекта.

Структура:
- `ФРОНТЭНД` — все отчеты и описания интерфейса
- `БЭКЭНД` — все отчеты и описания логики/данных

Новые описания добавляются в обе локации:
1. `docs` (рабочая папка)
2. `descriptions` (архив описаний по типам)
