Описание Contact Manager v0.18.2 (БЭКЭНД)

Изменения логики и системных компонентов:
1. `AppEventLogger` переработан:
   - расширенное логирование сессии (start/resume/stop/close, счетчики контактов, устройство);
   - анонимизация чувствительных фрагментов в сообщениях логов;
   - автоочистка логов старше 3 дней;
   - фиксация pending crash-отчета.
2. После падения приложение подготавливает email-отчет на следующем запуске:
   - вложение log-файла;
   - устройство, время начала и конца лог-сессии.
3. Подключен `FileProvider` для безопасной передачи log-файлов через почтовые клиенты.
4. Полный сброс данных изменен:
   - логи больше не очищаются;
   - после сброса сохраняется только группа `other` (в UI остаются «Все» + «Другое`).
5. Добавлен `PhoneNumberFormatter`-поток для исправления маски и нормализации телефонов.
6. В `ContactEditorActivity` реализовано авто-создание группы по вводу в поле «Другое».
7. Обновлены версии:
   - versionName: 0.18.2
   - versionCode: 26
   - update_sequence: 25

Проверка:
- assembleDebug, lintDebug, testDebugUnitTest выполнены успешно.
