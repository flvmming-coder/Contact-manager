Описание версии: Contact Manager v0.14.3 (PRE-RELEASE)
Тип: БЭКЭНД

Причина обновления:
Нужно добавить автоматическое уведомление об обновлениях и унифицировать логику сравнения версий для release и pre-release.

Что изменено в логике:
1. Добавлен UpdateChecker:
- запрос списка GitHub releases,
- отбор актуальной версии среди release и pre-release,
- нормализация/сравнение версий.
2. Добавлен UpdateCheckWorker + UpdateWorkScheduler:
- периодическая фоновая проверка обновлений,
- отправка системного уведомления при новой версии,
- защита от повторных уведомлений для одной и той же версии.
3. Добавлен UpdateNotificationHelper:
- создание notification channel,
- отправка уведомления с текстом:
  "Обновите до новой версии, чтобы приложение работало корректно".
4. Добавлено разрешение POST_NOTIFICATIONS и runtime-запрос в MainActivity.
5. Модель Contact расширена полем avatarColor, сохранение/чтение обновлено в ContactPrefsStorage.
6. Реализована палитра цветов AvatarColorPalette и ручной выбор цвета аватара в редакторе контакта.

Технический статус:
- assembleDebug: успешно,
- lintDebug: успешно,
- testDebugUnitTest: успешно.
