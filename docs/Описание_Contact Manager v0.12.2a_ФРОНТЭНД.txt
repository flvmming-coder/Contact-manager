Описание версии: Contact Manager v0.12.2a (TEST)
Тип: ФРОНТЭНД

Причина обновления:
В версии 0.12.2 на пользовательском устройстве наблюдался вылет при запуске.

Что сделано в UI для стабилизации:
1. Сохранен новый визуальный стиль 0.12.2 (градиенты, FAB, лента групп).
2. Упрощен рендер иконки в шапке для снижения риска InflateException.
3. Карточки контактов переведены на безопасный вариант разметки:
- скругленные углы,
- тонкая окантовка,
- без использования компонента, который мог конфликтовать с темой.
4. Добавлен аварийный стартовый экран:
- если на инициализации главного экрана возникает исключение, приложение не закрывается сразу,
- отображается текст о проблеме и кнопка "Повторить".

Дополнительно:
Добавлен механизм логирования событий интерфейса и действий пользователя для диагностики на реальном устройстве.
