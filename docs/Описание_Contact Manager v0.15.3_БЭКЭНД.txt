Описание версии Contact Manager v0.15.3 (19) — БЭКЭНД

Причина обновления:
Нужно было усилить механизм доставки обновлений, перенести управление ручным цветом в dev-контур и расширить импорт данных контактов.

Что изменено в логике:
1. Обновление и уведомления:
- обновлён `UpdateChecker` для получения прямой ссылки на APK из release assets;
- уведомление об обновлении сделано постоянным (`ongoing`) и дополнено action-кнопкой скачивания;
- в админ-панели проверка обновлений запускает прямую загрузку APK через `DownloadManager`.
2. Ролевая логика цвета аватара:
- добавлен dev-флаг ручного выбора цвета;
- в обычном режиме редактирования сохраняется только случайный цвет (без ручной палитры).
3. Импорт контактов с устройства:
- подтягивается `PHOTO_URI` и сохраняется в `avatarPhotoUri`;
- логика импорта имен/фамилий обновлена для сохранения многочастных строк без обрезки.
4. Developers-диалог:
- реализованы кликабельные `ClickableSpan` с переходом по VK-ссылкам.
5. Мультивыбор контактов:
- реализован режим выбора с удержанием 2 секунды;
- после активации — переключение выбора одним тапом;
- реализованы массовые операции: назначение группы и массовое удаление.
6. CI/release:
- workflow обновлены для релизной цепочки `0.15.2 -> latest`, `0.15.3 -> pre-release`;
- добавлена поддержка токена `FLVMMING_CODER_TOKEN` для публикации релизов от имени владельца.

Проверка:
Сборка, lint и unit-test выполнены успешно.
