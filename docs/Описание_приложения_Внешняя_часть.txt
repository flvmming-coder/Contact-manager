ВАРИАНТ 1. ФРОНТЭНД (ВНЕШНЯЯ ЧАСТЬ) — CONTACT MANAGER v0.11

Общая концепция
В версии v0.11 мобильный интерфейс переработан под визуальный стиль HTML-макета: градиентная шапка, компактная карточная сетка контактов, горизонтальные фильтры-группы и плавающая кнопка добавления.

Ключевые элементы UI
- Верхняя зона со статус-баром (время + системные индикаторы).
- Градиентный header с названием "Contact Manager".
- Кнопки действий в header: импорт, рабочие задачи, Google auth (mock).
- Плашка статистики: всего контактов, рабочих, импортированных.
- Поисковая строка в стиле pill.
- Горизонтальные вкладки-группы: Все, Семья, Друзья, Работа, Прочие.
- Карточки контактов с инициалами, телефоном, группой и кнопками редактирования/удаления.
- Маркер рабочего контакта (badge W).
- Плавающая кнопка (FAB) для создания контакта.

Поведение интерфейса
1. Поиск применяется мгновенно при вводе.
2. Переключение группы обновляет список без перезагрузки экрана.
3. Нажатие по карточке открывает редактирование.
4. При пустом результате отображается состояние "Нет контактов".
5. Импорт из текста поддерживает формат строк "Имя,Телефон,Email".

Экран редактирования
- Поля: имя, фамилия, телефон, email, группа, адрес, дата рождения.
- Флаг рабочего контакта.
- Поле задачи доступно только при включенном флаге рабочего контакта.
- Кнопки: сохранить/отмена.

Нефункциональные требования фронтэнда
- Совместимость UI с Android 5.0+ (API 21+).
- Время реакции на фильтрацию/поиск в пределах пользовательского ожидания (сотни миллисекунд).
- Единый стиль компонентов и читаемая структура экрана на телефонах.
