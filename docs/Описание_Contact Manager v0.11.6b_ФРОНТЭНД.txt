Описание версии: Contact Manager v0.11.6b
Тип: ФРОНТЭНД

Причина обновления:
После добавления расширенной формы контакта потребовалось обновить визуальную структуру главного экрана и сделать быстрый доступ к добавлению контакта.

Изменения интерфейса:
1. Кнопка «Добавить контакт» заменена на круглую FAB-кнопку в правом нижнем углу.
2. В FAB используется символ «плюс» для быстрого добавления контакта.
3. Верхний блок приложения переработан:
- слева размещена иконка контактов,
- заголовок Contact Manager отображается белым цветом,
- в блок включены счетчики общего количества и импортированных контактов,
- фон блока: градиент слева направо (#1e293b -> #0f172a).
4. Использована иконка контактов, адаптированная из SVG-коллекции (`svgs-full/solid/address-book.svg`).
5. В форме редактирования/добавления отображаются поля:
- имя,
- фамилия,
- телефон,
- email,
- адрес,
- дата рождения (вручную или через календарь).

Результат:
Интерфейс стал ближе к мобильному паттерну (FAB + информативная шапка), при этом функциональность формы расширена.
