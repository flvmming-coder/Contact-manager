Описание версии: Contact Manager v0.13 (TEST)
Тип: БЭКЭНД

Причина обновления:
Нужно было убрать прямые кнопки действий с карточки и перевести открытие редактора на жест, сохранив стабильность и предсказуемое поведение.

Что изменено в логике:
1. Переписан adapter карточек контактов.
2. Добавлена функция генерации инициалов из имени и фамилии с обработкой коротких/пустых значений.
3. Добавлен выбор цвета аватарки из палитры приложения по детерминированному индексу (на основе id контакта).
4. Реализована логика двойного нажатия:
- хранится id последнего нажатого контакта,
- хранится время последнего нажатия,
- если второе нажатие по тому же контакту в пределах 1000 мс -> открывается редактирование,
- иначе состояние ожидания обновляется без действий.

Технический статус:
- версия приложения обновлена до 0.13,
- сборка debug и статические проверки проходят.
