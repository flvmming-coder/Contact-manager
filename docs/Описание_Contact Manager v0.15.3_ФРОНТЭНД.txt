Описание версии Contact Manager v0.15.3 (19) — ФРОНТЭНД

Причина обновления:
Потребовалось унифицировать служебные окна приложения, улучшить UX сценариев обновления и добавить полноценный мультивыбор контактов для массовых операций.

Что обновлено:
1. Окна «Настройки» и «Админ-панель» получили единый визуальный стиль.
2. Кнопки в служебных окнах и системные диалоговые кнопки приведены к общему градиентному стилю `#f59e0b -> #fbbf24`.
3. Диалог создания/редактирования контакта стилизован под общий дизайн (скругленные поля/карточка, единые кнопки).
4. Ручной выбор цвета аватара скрыт для обычного режима и доступен только в админ-панели.
5. Добавлен экранный UX для мультивыбора контактов:
- первое выделение удержанием 2 секунды;
- дальнейший выбор по обычному нажатию;
- визуальное выделение карточек зелёной рамкой и полупрозрачным фоном;
- панель массовых действий (назначение группы/удаление/сброс).
6. Developers вынесены в отдельное окно с кликабельными ссылками VK.
7. Уведомление о новой версии дополнено явной кнопкой «Скачать».

Результат:
Интерфейс стал более цельным и предсказуемым, а сценарии массовой работы с контактами и обновлениями — быстрее для пользователя.
