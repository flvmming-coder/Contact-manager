Contact Manager v0.11 — ФРОНТЭНД

Назначение
Версия v0.11 переносит визуальный стиль HTML в нативный Android-интерфейс и развивает пользовательский поток.

Ключевые элементы UI
- Статус-бар с временем.
- Градиентный header с названием "Contact Manager".
- Кнопки действий: импорт, рабочие задачи, Google-auth mock.
- Блок статистики.
- Поисковый блок в стиле pill.
- Горизонтальные вкладки-группы.
- Карточки контактов с аватаром, группой и действиями.
- FAB для добавления нового контакта.

Поведение
1. Мгновенный поиск по имени/фамилии/телефону.
2. Фильтрация по группам с обновлением счетчиков.
3. Открытие редактирования по нажатию на карточку.
4. Отображение empty-state при пустом результате.

Статус
Интерфейс готов для пользовательского тестирования на Android 5.0+.
