DEMO v0 — ФРОНТЭНД

Назначение
Нулевая DEMO-версия предназначена для демонстрации пользовательского интерфейса и основных сценариев работы с контактами.

Ключевые элементы UI
- Мобильный контейнер в стиле экрана смартфона.
- Верхняя статусная зона.
- Градиентный header с названием экрана контактов.
- Блок статистики: всего контактов, рабочих, импортированных.
- Поисковая строка.
- Вкладки групп (все/семья/друзья/работа/прочие).
- Карточки контактов с быстрыми действиями.
- FAB-кнопка добавления.
- Модальные окна для создания/редактирования и импорта.

Поведение
1. Поиск выполняется в реальном времени.
2. Фильтрация по группам переключает контент без перезагрузки.
3. Карточка контакта поддерживает редактирование/удаление.
4. Пустое состояние выводится при отсутствии результатов.

Статус
Версия ориентирована на презентацию UX/UI и общей концепции приложения.
