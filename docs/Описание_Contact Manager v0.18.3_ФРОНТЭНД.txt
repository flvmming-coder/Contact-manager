Contact Manager v0.18.3 (26)
ФРОНТЭНД

Причина обновления:
- требовалась доработка UX в админ-панели, корзине и теме приложения.

Основные изменения интерфейса:
1) Developers:
- заменен текст проекта на: Non-commercial project for the NUST MISIS branch in Novotroitsk;
- добавлен визуальный отступ (пустая строка) перед блоком Support with dev.

2) Админ-панель:
- в окно логов добавлена кнопка «Очистить файлы» рядом с «Отмена»;
- при очистке открывается окно ввода PIN.

3) Темы:
- в настройках темы теперь 2 раздела:
  - режим темы: светлая / темная / системная;
  - цвет акцента активных кнопок: желтая, светло-серая, сиреневая, сине-голубая, пользовательская.
- пользовательская тема задается двумя HEX-цветами.

4) Корзина:
- удален выбор 7/30 дней, хранение фиксировано на 30 дней;
- удаленный контакт открывается по двойному нажатию;
- карточка удаленного контакта отображается только для просмотра (без редактирования).

5) Обновление приложения:
- после загрузки APK автоматически открывается системный установщик.
