Описание версии: Contact Manager v0.11.5b
Тип: БЭКЭНД

Причина обновления:
В 0.11.5a фиксировался неизвестный сбой при старте. Для снижения риска приложение переписано на простой и предсказуемый поток данных.

Архитектура и хранение:
1. Используется локальное хранилище SharedPreferences.
2. Контакты сохраняются в JSON-массиве.
3. Добавлен безопасный парсер с обработкой исключений, чтобы некорректные данные не вызывали падение.
4. Добавлена миграция: если в старых данных есть firstName/lastName, они объединяются в поле name.

Базовая логика:
- предустановленные группы: Семья, Друзья, Работа, Другое,
- предзаполненные тестовые контакты при первом запуске,
- операции CRUD: создать, прочитать, обновить, удалить.

Технический результат:
- минимизирована вероятность InflateException за счет удаления сложной разметки и нестабильных элементов UI,
- код упрощен для последующей отладки и расширения.
