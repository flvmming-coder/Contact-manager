Описание версии: Contact Manager v0.11.5b
Тип: ФРОНТЭНД

Причина обновления:
Версия 0.11.5a на ряде устройств (включая Android 13) завершалась аварийно при запуске. Основная цель v0.11.5b — полностью упростить интерфейс и убрать нестабильные элементы разметки.

Что изменено:
1. Интерфейс переписан с нуля в минимальном стиле.
2. Экран содержит только:
- список контактов,
- фильтр по группам,
- кнопку добавления контакта.
3. Карточка контакта теперь простая: имя, телефон, группа, кнопки "Редактировать" и "Удалить".
4. Создание и редактирование выполняются в одном диалоговом окне.

Функции пользователя:
- запуск приложения без сложной графики,
- просмотр контактов,
- фильтрация по группам,
- добавление/редактирование/удаление контакта.

Планы на следующую итерацию:
- добавить подтвержденное тестирование на Android 13/12/10,
- после стабилизации вернуть улучшения UI поэтапно.
