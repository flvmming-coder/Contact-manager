Описание версии: Contact Manager v0.11.6a
Тип: БЭКЭНД

Причина обновления:
Для поддержки новой UI-логики (статистика и поиск) потребовалось расширить модель данных и поток обработки контактов.

Что реализовано в логике и данных:
1. Модель контакта расширена полем isImported.
2. Хранилище сохраняет и читает признак импорта для каждого контакта.
3. Добавлен расчет статистики:
- общее количество контактов,
- количество контактов с признаком импорта.
4. Добавлена фильтрация списка по поисковому запросу:
- совпадение по имени,
- совпадение по номеру телефона.
5. Поиск работает совместно с фильтром по группе.

Технический результат:
Приложение поддерживает русифицированный интерфейс, корректную статистику и быстрый поиск по контактам без усложнения архитектуры.
