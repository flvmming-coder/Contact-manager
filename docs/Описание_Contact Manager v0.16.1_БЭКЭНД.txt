Описание Contact Manager v0.16.1 (БЭКЭНД)

Изменения логики и данных:
1. ContactPrefsStorage расширен поддержкой настраиваемых групп:
   - хранение списка групп в prefs;
   - создание/переименование/удаление групп;
   - специальная группа «Без группы» (ungrouped).
2. При удалении группы контакты автоматически переводятся в «Без группы», без удаления самих контактов.
3. Импорт из телефонной книги расширен:
   - чтение memberships через ContactsContract.CommonDataKinds.GroupMembership;
   - создание отсутствующих групп по названиям с устройства;
   - импорт контакта сразу в соответствующую группу.
4. ContactAdapter и item_contact обновлены для буквенных секций в списке.
5. AppEventLogger расширен методами чтения/перечня логов для открытия в админ-панели.
6. Версии обновлены:
   - versionName: 0.16.1
   - versionCode: 22
   - update_sequence: 21

Проверка:
- assembleDebug, lintDebug, testDebugUnitTest выполнены успешно.
