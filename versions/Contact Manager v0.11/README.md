# Contact Manager v0.11

Версия Android-приложения, переработанная по HTML v0.11.

Содержимое:
- `Мобильное_v0.11.html` — исходный HTML-макет v0.11.
- `ContactManager-v0.11-debug.apk` — debug APK для установки и тестирования.

Основные функции:
- Новый визуальный стиль в духе HTML v0.11
- Поиск и фильтрация по группам
- CRUD для контактов
- Импорт из текста и Google mock
- Локальное хранение в SQLite
