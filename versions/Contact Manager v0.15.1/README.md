# Contact Manager v0.15.1 (PRE-RELEASE)

Причина обновления:
- версия 0.14.4 стабильна, но требовалась доработка визуальной части аватаров;
- добавлена пользовательская настройка светлой/тёмной темы с сохранением;
- добавлен фирменный логотип приложения для иконки запуска.

Что изменено:
1. Аватары контактов и превью в форме редактирования теперь со скруглёнными углами в стиле всей темы приложения.
2. В разделе «Настройки» добавлен пункт «Сменить тему»:
   - Светлая (по умолчанию)
   - Тёмная
3. Выбор темы сохраняется в локальных настройках и применяется при следующих запусках приложения.
4. Обновлена иконка приложения (launcher icon): логотип контактов с круглой градиентной обводкой `#f59e0b -> #fbbf24`.
5. Логотип в шапке приложения переведён на такой же стиль круглой градиентной обводки.
6. Поддержка тёмной темы расширена через `DayNight` и night-ресурсы цветов.

- `:app:testDebugUnitTest` — успешно
