# Contact Manager v0.15.2 (RELEASE)

Причина обновления:
- в 0.15.1 выбор цвета аватара был доступен в обычном режиме редактирования;
- при тёмной теме часть текста в админ-панели читалась плохо;
- требовалась доработка импорта контактов (фото + полные имена) и унификация стиля формы редактирования.

Что изменено:
1. Ручной выбор цвета аватара перенесён в админ-панель:
   - для обычного пользователя оставлен только случайный цвет;
   - в админ-панели добавлен переключатель, который включает/выключает ручной выбор цвета.
2. В обычном режиме в форме контакта оставлены только действия по фото:
   - выбрать фотографию,
   - удалить фотографию.
3. Импорт контактов с устройства теперь подтягивает фото контакта, если оно есть.
4. При импорте имён с несколькими словами сохраняются все слова (без обрезки до первого слова).
5. Улучшена читаемость в тёмной теме:
   - поправлены цветовые ресурсы и текстовые цвета в админ-панели и формах.
6. Информация о разработчиках вынесена в отдельную кнопку в админ-панели:
   - открывается отдельный диалог Developers;
   - имена разработчиков кликабельны и ведут на VK:
     - Kurenkov E. E. → https://vk.com/lxrdx
     - Kozin S. V. → https://vk.com/id225880613
7. Диалог создания/редактирования контакта приведён к единому стилю:
   - скруглённые поля ввода,
   - единый фон диалога,
   - кнопки в градиенте `#f59e0b -> #fbbf24`.
