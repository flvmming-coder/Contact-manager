# Contact Manager v0.14.4 (RELEASE)

Причина обновления:
- доработать карточку и редактирование контакта под расширенные требования (кастомный цвет + фото-аватар),
- перенести импорт в меню настроек и добавить экспорт VCF,
- добавить отдельную точку входа в Google-интеграцию.

Что реализовано:
1. В редактировании контакта добавлен выбор цвета аватара в 3 режимах:
   - случайный цвет,
   - выбор из расширенной палитры,
   - ручная настройка цвета через RGB-палитру.
2. Пользователь может нажать на круг предпросмотра цвета и открыть палитру выбора.
3. Добавлен выбор фото для аватара контакта из доступных галерей/провайдеров устройства.
4. Фото аватара сохраняется у контакта и отображается в карточке вместо буквенной заглушки.
5. В верхней панели вместо старой кнопки импорта размещены две кнопки:
   - настройки (шестеренка),
   - Google.
6. В настройках добавлены действия:
   - импорт контактов из телефонной книги,
   - экспорт всех контактов в формат .vcf,
   - полная очистка списка контактов.
7. В Google-диалоге добавлено предложение включить сетевой режим через подключение Google-аккаунта.
8. Счётчик версии в админ-панели обновлён до: `0.14.4 (16)`.
