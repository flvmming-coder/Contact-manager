# Contact Manager v0.18.5 (PRE-RELEASE)

Причина обновления:
- в 0.18.4 нужно было доработать админ-управление и сценарии обязательного перезапуска после критичных действий;
- требовалось перевести выбор пользовательской темы из HEX-ввода на визуальную палитру.

Что сделано:
1. Админ-панель:
   - добавлено поле ввода кода `1410`;
   - при успешном вводе включается bypass: можно пропускать принудительный перезапуск до конца текущего сеанса.
2. Группа «Служебные»:
   - флаг показа группы сбрасывается при новом запуске приложения;
   - группа снова появляется только после повторного входа в админ-панель.
3. Логи:
   - переведены на rolling-режим: один файл на 3 дня (`rolling-<window>.txt`);
   - одновременно хранятся текущий и предыдущий период, более старые файлы удаляются автоматически.
4. Перезапуск после важных операций:
   - после импорта контактов и после смены темы показывается обязательный диалог (30 сек + кнопка OK);
   - по таймеру или по OK приложение полностью закрывается для ручного перезапуска;
   - при активном коде `1410` перезапуск можно пропустить.
5. Темы:
   - акцент применяется шире: FAB, вкладки групп, верхние иконки, счетчики, кнопки в админ-панели;
   - пользовательская тема теперь настраивается через визуальную палитру двух границ градиента (без HEX-ввода).
