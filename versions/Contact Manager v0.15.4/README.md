# Contact Manager v0.15.4 (RELEASE)

Причина обновления:
- ускорить вход в режим выбора контактов;
- убрать визуальные конфликты при массовом выборе (FAB + панель действий);
- уменьшить акцентность функциональных градиентных кнопок.

Что изменено:
1. Удержание для первого выбора контакта уменьшено с 2 секунд до 1 секунды.
2. Во время режима выбора контактов кнопка "+" (FAB) автоматически скрывается.
3. Панель массовых действий переработана:
   - адаптивная компоновка по ширине экрана;
   - кнопки размещаются в ряд с равной шириной;
   - больше не перекрываются FAB.
4. Градиентные функциональные кнопки уменьшены по визуальному весу:
   - уменьшены высота, паддинги и размер текста;
   - применено к системным кнопкам диалогов (Отмена/Сохранить/Подключить и т.д.);
   - применено к кнопкам Календарь/Аватар/Настройки/Админ-панель.
