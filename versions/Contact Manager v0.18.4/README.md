# Contact Manager v0.18.4 (RELEASE)

Причина обновления:
- после установки 0.18.3 приложение не запускалось на ряде устройств (в том числе Android 13);
- требовалось устранить критический сбой старта и стабилизировать стартовую инициализацию.

Что исправлено:
1. Критический фикс запуска:
   - добавлена безопасная миграция `theme_mode` из старого формата (`Int`) в новый (`String`);
   - устранен риск `ClassCastException` при старте приложения после обновления.
2. Защитный запуск темы:
   - применение темы обернуто безопасным fallback;
   - при ошибке применяется дефолтная тема без краша приложения.
3. Логи/краши на раннем этапе:
   - инициализация логгера перенесена раньше применения темы в `Application`;
   - при ошибках ранней инициализации событие теперь записывается в лог.
4. Очистка кода:
   - удален неиспользуемый устаревший код диалогов настроек/корзины в `MainActivity`;
   - оставлены только актуальные маршруты через отдельные экраны.
