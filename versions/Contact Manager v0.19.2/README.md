# Contact Manager v0.19.2 (RELEASE)

Причина обновления:
- в 0.19.1 при QR-переносе кириллица могла повреждаться (вместо русских символов отображались `?`).

Что сделано:
1. Исправление кодировки QR:
- генерация QR переведена на UTF-8 с явными подсказками кодека;
- добавлен fallback-декодер для восстановления mojibake-строк при сканировании.

2. Вертикальный QR-сканер:
- добавлена отдельная `CaptureActivity` в портретной ориентации;
- сканер теперь открывается вертикально.

3. Логика кнопки сканирования:
- для существующего контакта кнопка «Сканировать QR-код» скрыта;
- кнопка остаётся только в сценарии создания нового контакта.

4. Переработка блока аватара в редакторе:
- аватар перенесён вверх формы и выровнен по центру;
- размер аватара увеличен;
- кнопки «Выбрать фото» / «Убрать фото» вынесены под аватар.
