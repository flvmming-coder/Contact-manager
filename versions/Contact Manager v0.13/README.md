# Contact Manager v0.13 (TEST)

Цель обновления:
- перейти к более компактной карточке контакта,
- убрать кнопки на карточке и сделать редактирование по двойному нажатию,
- добавить визуальный блок аватара-заглушки.

Что изменено:
- в карточке слева добавлена квадратная аватарка с инициалами,
- инициалы формируются так:
  - имя + фамилия: первая буква имени + первая буква фамилии,
  - без фамилии: первые 2 буквы имени,
  - если в имени только 1 буква: показывается 1 буква,
- фон аватарки подбирается автоматически из палитры темы,
- справа от аватарки: ФИО жирным и телефон с иконкой телефона,
- кнопки "Редактировать" и "Удалить" удалены с карточки,
- редактирование открывается только при двойном тапе по карточке в течение 1 секунды.
