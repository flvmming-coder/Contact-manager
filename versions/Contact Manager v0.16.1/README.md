# Contact Manager v0.16.1 (PRE-RELEASE)

Причина обновления:
- нужно расширить управление группами внутри приложения;
- при импорте из телефонной книги требовалось переносить и структуру групп;
- необходимо добавить буквенную навигацию в длинном списке контактов;
- в админ-панели требовался быстрый доступ к логам для чтения.

Что сделано:
1. Список контактов отображается по алфавиту с буквенными секциями (A/Б/В и т.д.) прямо в ленте.
2. Импорт контактов теперь учитывает группы устройства:
   - если у контакта есть группа в телефонной книге, в приложении создается такая же группа;
   - контакт импортируется с этой группой.
3. Реализовано полноценное управление группами:
   - создание новой группы;
   - переименование существующей;
   - удаление группы.
4. При удалении группы контакты не удаляются:
   - остаются в списке «Все»;
   - их принадлежность к удаленной группе сбрасывается («Без группы»).
5. В строку групп добавлена иконка шестеренки для открытия управления группами.
6. В админ-панели добавлена кнопка «Открыть файлы логов»:
   - отображает список лог-файлов;
   - открывает выбранный лог внутри приложения для чтения.
