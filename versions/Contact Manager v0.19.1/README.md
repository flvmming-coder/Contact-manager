# Contact Manager v0.19.1 (PRE-RELEASE)

Причина обновления:
- в версии 0.18.7 отсутствовал офлайн-перенос одного контакта между устройствами без интернета.

Что сделано:
1. QR-перенос контакта (ключевая функция):
- на карточку контакта добавлена отдельная кнопка QR рядом с «Избранным»;
- для каждого контакта генерируется персональный QR-код;
- на экране создания/редактирования добавлена кнопка «Сканировать QR-код»;
- после сканирования данные заполняют форму, пользователь может отредактировать контакт перед сохранением;
- принадлежность к группе через QR не переносится (по ТЗ).

2. Экран редактирования контакта:
- при открытии существующего контакта (двойной тап из списка) показывается кнопка «Сгенерировать QR-код» внизу формы.

3. Иконки и UI:
- текстовый символ сердца заменён на SVG-иконку `heart.svg`;
- добавлена отдельная иконка QR на плитке контакта;
- новые кнопки подчиняются выбранной теме приложения.

4. Доступ к камере:
- добавлен запрос разрешения `CAMERA` для QR-сканирования.

5. Темы для дальтонизма:
- цветокоррекция режимов (дейтераномалия/протаномалия/тританомалия/монохром) теперь применяется и к фото-аватаркам контактов.
