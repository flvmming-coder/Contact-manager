# Contact Manager v0.17 (RELEASE)

Источник сборки:
- техническая сборка v0.17.1 (23), переведена в стабильный релизный канал.

Что зафиксировано в релизе:
1. Полноэкранные рабочие вкладки вместо ключевых диалогов:
   - создание/редактирование контакта;
   - настройки;
   - управление группами.
2. Индикатор прогресса импорта в шапке главного экрана.
3. Обязательное окно перезапуска после импорта (30 секунд + кнопка мгновенного сворачивания).
4. Расширенное логирование пользовательских и системных событий.
