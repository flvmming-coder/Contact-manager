# Contact Manager v0.18.7 (RELEASE)

Причина обновления:
- после 0.18.6 потребовались правки UX/темизации и финализация логики «Избранное» без потери основной группы контакта.

Что сделано:
1. Админ-панель:
   - перестроен порядок блока: переключатель логов -> путь логов -> версия;
   - статус bypass оставлен в формате `Activated` / `Disactivated`.
2. Developers:
   - цвет ссылок теперь совпадает с цветом строки статистики контактов/импорта (текущий тематический акцент).
3. Прогресс-бар импорта:
   - цвет теперь динамически подстраивается под выбранную тему/палитру.
4. Избранное (ключевой фикс):
   - контакт при пометке сердцем остается в своей основной группе;
   - добавлена независимая принадлежность к «Избранному» через флаг;
   - фильтр «Избранное» работает по этому флагу;
   - в редакторе/создании контакта «Избранное» недоступно в списке групп.
5. Темы:
   - пункт переименован в «Основная тема»;
   - добавлены режимы для дальтонизма:
     - Дейтераномалия
     - Протаномалия
     - Тританомалия
     - Монохром
6. Иконка «Избранное»:
   - увеличена до размера, сопоставимого с иконкой шестеренки групп;
   - сделана в виде сердца в скругленном квадрате-рамке;
   - цвета: активный `#be123c`, неактивный `#334155` (light) / `#a3a5bd` (dark).
