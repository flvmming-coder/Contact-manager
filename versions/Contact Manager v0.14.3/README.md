# Contact Manager v0.14.3 (RELEASE)

Цель обновления:
- добавить уведомление о новых обновлениях,
- проверять не только релизы, но и pre-release,
- расширить админ-панель и палитру цветов аватаров.

Что добавлено:
1. Фоновая проверка обновлений (release + pre-release) через GitHub API.
2. Push-уведомление (локальное системное уведомление Android) при обнаружении новой версии:
   "Обновите до новой версии, чтобы приложение работало корректно".
3. В админ-панели добавлен блок с разработчиками:
   "Разработчики: Куренков Е. Е., Козин С. В.
    Новотроицк, МИСИС".
4. В карточках контактов расширена палитра фоновых цветов аватаров по заданному списку.
5. В редактировании контакта добавлен выбор цвета аватара через палитру.
6. Проверка обновлений в админ-панели теперь учитывает и последний pre-release.
