ДЕМО ВЕРСИЯ v0 — ОПИСАНИЕ ФРОНТЭНДА (ИСХОДНОЕ)

Назначение
Начальная DEMO-версия предназначена для демонстрации пользовательского сценария работы с контактами в современном мобильном интерфейсе. Основной фокус версии — визуальная презентация и базовые интерактивные действия.

Состав интерфейса
- Мобильный контейнер в стиле экрана смартфона.
- Верхняя динамическая панель/статус-бар.
- Градиентный header с названием экрана "Контакты".
- Блок статистики: всего контактов, рабочих, импортированных.
- Поисковая строка.
- Горизонтальные вкладки групп (все/семья/друзья/работа/прочие).
- Список контактов карточками с аватаром, именем, телефоном и действиями.
- FAB-кнопка добавления контакта.
- Модальные окна для добавления/редактирования и импорта.
- Всплывающие уведомления (toast).

Поведение интерфейса
1. Контакты группируются по первой букве имени.
2. Поиск обновляет список в реальном времени.
3. Вкладки групп переключают фильтрацию.
4. Редактирование и удаление доступны из карточки.
5. Импорт можно выполнить из текстового списка/мок-сценариев.

Статус версии
- Версия ориентирована на демонстрацию UX/UI и общего потока работы.
- Интерфейс демонстрационный, но полностью интерактивный в рамках HTML-прототипа.
