# Contact Manager v0.19.3 (PRE-RELEASE)

Причина обновления:
- в 0.19.2 требовалось расширить перенос: аватарка внутри QR, массовый перенос и административный контроль БД.

Что сделано:
1. QR-перенос контакта:
- в payload добавлена аватарка контакта (сильное сжатие для QR);
- внедрён новый компактный формат `CM3` (gzip + base64 + короткие ключи), чтобы QR был легче для чтения на старых камерах.

2. Массовый перенос контактов:
- добавлен раздел «Передача всех контактов» в настройках;
- сценарии:
  - экспорт всех в QR (если пакет помещается в ограничение);
  - импорт всех по QR;
  - экспорт всех через Bluetooth/Share (файл);
  - импорт из Bluetooth-файла.

3. База данных (админ-контроль):
- добавлено зеркальное SQLite-хранилище контактов в таблице формата `DataBase_<IMEI/ID>`;
- при «очистке всей информации» зеркало БД не очищается;
- в админ-панели добавлены:
  - просмотр снапшота БД;
  - очистка БД только по PIN `0193`.
- при неверном PIN `0193` приложение аварийно завершает работу (по ТЗ).

4. Релизы:
- `v0.19.2` переключена в `release/latest`;
- `v0.19.3` опубликована как `pre-release`.
