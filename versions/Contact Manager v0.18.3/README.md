# Contact Manager v0.18.3 (PRE-RELEASE)

Причина обновления:
- в версии 0.18.2 требовалось расширить админ-инструменты и обновление приложения;
- требовалось исправить поведение группы «Другое» и доработать работу корзины;
- требовалось улучшить UX маски телефона при удалении цифр.

Что изменено:
1. Developers:
   - строка проекта заменена на `Non-commercial project for the NUST MISIS branch in Novotroitsk`;
   - добавлена пустая строка-разделитель перед `Support with dev`.
2. Обновления приложения:
   - после нажатия «Скачать обновление» APK скачивается и автоматически открывается системный установщик.
3. Админ-панель (логи):
   - добавлена кнопка «Очистить файлы»;
   - очистка защищена PIN-кодом `0183`;
   - при неверном PIN клавиатура закрывается и событие пишется в лог как попытка несанкционированного доступа.
4. Группы:
   - «Другое» стало виртуальной группой (только в редакторе контакта для создания новой группы);
   - «Другое» не показывается в фильтрах и в управлении группами;
   - устранено автосоздание дефолтных групп после удаления «Другое».
5. Корзина:
   - срок хранения зафиксирован на 30 дней;
   - открытие удаленного контакта по двойному нажатию;
   - просмотр данных в read-only режиме (для правок сначала восстановить контакт).
6. Темы:
   - разделены настройки на две ветки:
     - режим: светлая / темная / системная;
     - акцент кнопок: желтая / светло-серая / сиреневая / сине-голубая / пользовательская (2 HEX-цвета).
7. Маска телефона:
   - сохранен фиксированный префикс `+7`;
   - улучшено удаление цифр через разделители `() -` без ручного переставления курсора.
