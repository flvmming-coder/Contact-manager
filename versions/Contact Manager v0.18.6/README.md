# Contact Manager v0.18.6 (PRE-RELEASE)

Причина обновления:
- после 0.18.5 требовалось усилить защиту админ-части и доработать визуальные/UX-детали;
- необходимо было закрыть пробелы по применению темы на отдельных кнопках и добавить «Избранное».

Что сделано:
1. Админ-панель и защита кода 1410:
   - строка статуса упрощена до `Activated` / `Disactivated`;
   - при неверном вводе кода bypass полностью отключается и блокируется;
   - поле ввода PIN и кнопка применения скрываются;
   - повторное появление PIN-блока только после обновления приложения или после двух полных сбросов подряд.
2. Тема и кнопки:
   - исправлено применение темы для кнопок:
     - «Добавить» в управлении группами;
     - «Очистить корзину» в корзине;
     - «Сохранить», «Календарь», «Выбрать фото», «Убрать фото» в редакторе контакта;
   - добавлен автоматический выбор цвета текста на кнопках (светлый/темный по контрасту).
3. Группы:
   - группа «Служебные» защищена от изменения имени;
   - добавлена системная группа «Избранное» (создается автоматически по действию пользователя, имя не редактируется).
4. Корзина:
   - удаленный контакт открывается в том же полноэкранном редакторе, но в режиме только для чтения;
   - редактирование недоступно, доступен просмотр всей информации.
5. Избранное на плитке контакта:
   - добавлен индикатор-сердце в скругленном квадратике справа на карточке;
   - нажатие добавляет контакт в «Избранное» (повторное нажатие снимает);
   - цвета:
     - активный: `#be123c`;
     - неактивный: `#334155` (светлая тема), `#a3a5bd` (темная тема).
