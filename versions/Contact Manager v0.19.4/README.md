# Contact Manager v0.19.4 (PRE-RELEASE)

Причина обновления:
- после 0.19.3 потребовались улучшения админ-инструментов, сортировки и поиска по номеру в маске/без маски.

Что сделано:
1. Админ-панель:
- вход по удержанию шапки изменён с 5 до 2 секунд;
- добавлена выгрузка БД в `.xlsx` c моментальным шарингом файла.

2. Контакты и UI:
- на плитке контакта увеличены кнопки «Избранное» и «QR-код»;
- увеличено расстояние между этими кнопками.

3. Главный экран:
- добавлен выпадающий список сортировки:
  - Имя: А-Я;
  - Имя: Я-А;
  - Дата создания: сначала новые;
  - Дата создания: сначала старые.

4. Поиск и БД:
- поиск номера стал работать и по виду с маской (`41-02`), и по цифрам без маски (`4102`);
- в зеркале БД добавлено поле `phone_raw` с видом `+7XXXXXXXXXX` для РФ/КЗ.

5. Дата рождения:
- маска `dd/MM/yyyy` с автоподстановкой `/`;
- добавлена проверка на реальность даты (месяц/день/високосность).

6. Релизы:
- `v0.19.3` переведена в `latest release`;
- `v0.19.4` публикуется как `pre-release`.
