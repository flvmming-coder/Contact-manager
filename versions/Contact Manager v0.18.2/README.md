# Contact Manager v0.18.2 (PRE-RELEASE)

Причина обновления:
- расширение и усиление логирования для админ-панели;
- доработка сброса данных и групп после полного удаления;
- исправление маски ввода номера телефона и расширение сценария «Другое».

Что изменено:
1. Логи теперь фиксируют полный анонимный трек сессии:
   - запуск/возобновление/остановка/закрытие;
   - количество контактов и импортированных контактов;
   - устройство и версию ОС;
   - события ошибок и проверок обновлений;
   - факт полного сброса данных.
2. Логи анонимизированы:
   - email и длинные числовые последовательности маскируются перед записью.
3. Ротация логов:
   - лог-файлы старше 3 суток удаляются автоматически.
4. После непредвиденного сбоя формируется crash-отчет:
   - на следующем запуске открывается письмо разработчику с вложенным логом;
   - в письме: устройство, время начала/конца лог-сессии, log-файл.
5. В разделе «Developers» добавлена строка:
   - `Support with dev: flvmming.dev@gmail.com` (кликабельная).
6. Полный сброс данных:
   - логи больше не удаляются;
   - после сброса остаются только группы «Все» и «Другое».
7. При выборе группы «Другое» в редакторе контакта:
   - появляется поле ввода названия новой группы;
   - группа создается автоматически при сохранении/обновлении контакта.
8. Исправлена маска телефона:
   - фиксированный префикс `+7`;
   - пользователь вводит только цифры;
   - формат `+7 (XXX) XXX-XX-XX` подставляется автоматически;
   - при удалении удаляются только цифры, префикс `+7` сохраняется.
