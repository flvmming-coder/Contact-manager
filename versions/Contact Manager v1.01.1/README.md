# Contact Manager v1.01.1 (PRE-RELEASE)

Причина обновления:
- в 1.0.0 сортировка в тёмной теме была плохо читаемой (белый текст на светлом фоне);
- при передаче всех контактов один большой QR часто упирался в лимит размера.

Что изменено:
1. Сортировка:
- добавлены кастомные макеты spinner для выбранного элемента и выпадающего списка;
- цвета текста/фона теперь корректно подстраиваются под светлую и тёмную темы.

2. Передача контактов:
- внедрён режим передачи «QR-серия» для всех контактов;
- данные автоматически делятся на несколько QR-кадров;
- импорт принимает кадры последовательно, собирает сессию и объединяет контакты;
- при конфликте сессии пользователь получает предупреждение и перезапускает импорт.

3. Совместимость:
- старый сценарий «один большой QR» сохранён как fallback при сканировании;
- Bluetooth-сценарий для transfer-all убран из меню в пользу более универсального QR-потока.

Релизы:
- `v1.0.0` остаётся `latest release`;
- `v1.01.1` публикуется как `pre-release`.
