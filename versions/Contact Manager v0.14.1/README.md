# Contact Manager v0.14.1 (TEST)

Цель обновления:
- добавить пользовательский комментарий к контакту,
- добавить скрытую админ-панель для диагностики и контроля логов,
- добавить проверку новой версии по GitHub Release.

Что добавлено:
1. В редактировании/создании контакта добавлено поле "Комментарий".
   - ограничение: до 512 символов.
2. Скрытая админ-панель:
   - открывается 5 нажатиями за 2 секунды по шапке (логотип + "Contact Manager"),
   - переключатель включения/выключения логов,
   - отображение текущей версии приложения,
   - отображение пути сохранения логов,
   - проверка последней версии на GitHub.
3. Проверка интернета:
   - если интернет отключен, показывается предупреждающий диалог.
4. Логи:
   - формат .txt,
   - попытка сохранения в Android/data/com.contact.manager/ContactManagerLogs,
   - если путь недоступен, используется безопасный fallback директории приложения.

APK:
- `ContactManager-v0.14.1-debug.apk`
