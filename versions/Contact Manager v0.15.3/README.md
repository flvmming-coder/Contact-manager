# Contact Manager v0.15.3 (RELEASE)

Причина обновления:
- в версии 0.15.2 требовалась дальнейшая доработка механизма обновлений и UI-служебных окон;
- нужно было завершить сценарий массовой работы с контактами;
- необходимо перевести ручной выбор цвета в админ-режим и расширить импорт контактов.

Что сделано:
1. Уведомление об обновлении сделано постоянным (ongoing): его нельзя убрать свайпом.
2. В уведомление добавлена кнопка «Скачать», которая ведет на загрузку обновления.
3. В админ-панели проверка обновления теперь предлагает скачать APK сразу из приложения (через DownloadManager), без перехода на внешние страницы.
4. Раздел «Настройки» и «Админ-панель» приведены к единому стилю:
   - общий фон диалогов,
   - единые градиентные кнопки `#f59e0b -> #fbbf24`.
5. Системные кнопки в диалогах (Отмена/Сохранить/Удалить и т.д.) стилизованы в общий градиентный стиль.
6. Ручной выбор цвета аватара перенесен в админ-панель:
   - обычному пользователю оставлен случайный цвет;
   - ручной выбор включается переключателем в админ-режиме.
7. Импорт контактов улучшен:
   - импортируются фото контактов (если есть);
   - имена и фамилии с несколькими словами сохраняются полностью.
8. Developers вынесены в отдельную кнопку в админ-панели:
   - открывается отдельный диалог;
   - VK-профили разработчиков кликабельны.
9. Добавлен массовый выбор контактов:
   - первое выделение удержанием 2 секунды;
   - далее последующие контакты выбираются обычным нажатием;
   - выделенная карточка: зеленая рамка `#15803d` + фон `#648b67` с прозрачностью 30%;
   - доступны массовые действия: назначить группу и удалить выбранные.

- `:app:lintDebug` — успешно
- `:app:testDebugUnitTest` — успешно
