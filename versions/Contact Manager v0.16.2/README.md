# Contact Manager v0.16.2 (PRE-RELEASE)

Причина обновления:
- расширение импорта групп из телефонной книги;
- внедрение безопасного удаления через корзину;
- добавление полного сброса данных приложения.

Что изменено:
1. Импорт контактов из телефонной книги расширен по группам:
   - для контакта читаются названия его групп с устройства;
   - группы автоматически создаются в приложении с теми же названиями;
   - контакт импортируется с сохранением групповой принадлежности.
2. В настройках добавлена вкладка «Корзина»:
   - удаленные контакты отправляются в корзину;
   - контакты исчезают из активных групп/списков;
   - можно восстановить или удалить навсегда.
3. Срок хранения в корзине задается пользователем: 7 или 30 дней.
4. По истечении срока хранения контакт удаляется из корзины безвозвратно.
5. Кнопка «Удалить все контакты» заменена на «Удалить всю информацию»:
   - удаляются контакты, группы, корзина и служебные данные;
   - выводится предупреждение, что действие необратимо.
