# Contact Manager v0.14.2 (TEST)

Цель обновления:
- добавить счетчик номера обновления в админ-панель,
- добавить импорт контактов с устройства пользователя,
- изменить открытие админ-панели на удерживание 5 секунд.

Что реализовано:
1. Админ-панель:
- версия отображается в формате: `Версия приложения: 0.14.2 (14)`.
2. Импорт контактов с устройства:
- кнопка импорта добавлена в шапку,
- иконка кнопки взята из `svgs-full/solid/file-import.svg`,
- запрашивается разрешение `READ_CONTACTS`,
- импортируются поля: имя, фамилия, телефон, email, адрес, дата рождения (если доступна), комментарий/заметка,
- импортированные контакты помечаются как imported.
3. Вход в админ-панель:
- теперь открывается удержанием шапки (иконка + Contact Manager) в течение 5 секунд.

APK:
- `ContactManager-v0.14.2-debug.apk`
