# Contact Manager v0.11.6b

Changes:
- Extended contact form: first name, last name, phone, email, address, birthday.
- Optional email now has validation (`@` is required if email is entered).
- Birthday supports manual input (`dd/mm/yyyy`) and calendar picker.
- Added strict birthday validation (real calendar date only).
- Added input formatter for birthday to split ambiguous input (e.g. `99` -> day `9`, month `9`).

APK:
- `ContactManager-v0.11.6b-debug.apk`
