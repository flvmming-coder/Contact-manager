# Contact Manager v0.17.1 (PRE-RELEASE)

Причина обновления:
- перейти от всплывающих диалогов к полноэкранным рабочим вкладкам;
- сделать импорт более прозрачным и безопасным для пользователя;
- усилить диагностические логи для администраторов и разработчиков.

Что изменено:
1. Взаимодействие через полноэкранные экраны вместо диалогов:
   - создание/редактирование контакта (`+`) открывается как отдельная вкладка на весь экран;
   - «Настройки» открываются как отдельная вкладка;
   - «Управление группами» открывается как отдельная вкладка.
2. Иконка шестеренки управления группами уменьшена до ~0.8 от предыдущего размера.
3. В шапку главного экрана добавлен progress-bar импорта контактов:
   - показывается только во время процесса импорта;
   - скрывается после завершения.
4. После импорта контактов добавлено обязательное окно перезапуска:
   - запускается таймер на 30 секунд с авто-сворачиванием приложения;
   - кнопка «ОК» сворачивает приложение сразу, без ожидания таймера;
   - если админ-панель была активирована до импорта, автосворачивание не выполняется.
5. Логирование расширено:
   - фиксируются ключевые переходы по новым экранам;
   - фиксируются события импорта, перезапуска, ошибок и системных операций.
